# Artifact Pipeline

Simple Python application used as the source project for the artifact pipeline lab.

## Contents

```
├── src/
│   └── app.py
└── trigger.txt
```

## Run locally

```bash
python src/app.py
```

The app version is `1.0.0`. The `build()` helper in `src/app.py` can create `dist/version.txt` if you want to test the build output locally.
